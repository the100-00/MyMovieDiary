# My Movie Diary 🎬

20-2학기 모바일 응용 과목에 제출했던 개인 프로젝트입니다.<br>
TheMovieDB API에서 영화 정보를 추출해 내가 감상한 영화를 기록할 수 있고, <br>
Google Map API를 사용하여 현재 위치 정보와 주변 영화관을 검색할 수 있습니다. <br>
카카오톡 API를 이용해 앱 링크 공유도 가능하게끔 했습니다.<br>
